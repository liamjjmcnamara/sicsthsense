Detailed Description

An interface that classes may implement when requesting a sensor scan. The SensorsManager will notify this interface when events occur during the scan. There are three events that may occur: Scan started, scan finished, and new sensor found. Scan started and scan finished events are signalled using SensorScanEvents#sensorScanStarted and SensorScanEvents#sensorScanFinished, When a new sensor is discovered, SensorScanEvents#newSensorFound(SensorBase, boolean) is called.
